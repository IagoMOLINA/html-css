Adicionando Emojis

Para usar emojis no HTMl temos que pesquisar o valor dele em hexadical e adicionarmos antes do valor: &#x + o valor do emoji

    Lista de Emojis
  • Emoji da mãozinha como esse 🖖 o código é 1F596
  • Emoji do óculos como esse 🤓 o código é 1F913
